FOCO DUPLO DIGITAL - SITE SUPER PREMIUM

Base usada:
- Código do pacote foco-duplo-super-premium-cards-clicaveis.

O que foi adicionado do trabalho anterior:
- Fotos reais da Cacauê no portfólio e galeria de fotografia.
- Cards clicáveis para Ana, Carlos, Gilliel e Isa.
- Galeria individual para cada pessoa.
- Lightbox para ampliar fotos.
- Textos com acentuação corrigida.
- Duplicações de modais removidas.
- Pacotes sem preço fixo, usando "Sob consulta".
- Botão Área admin com login via Supabase Auth.
- Admin liberado apenas para os e-mails configurados no admin.js.

Arquivos principais:
- index.html
- style.css
- script.js
- assets/

Como abrir:
1. Abra o arquivo index.html no navegador.
2. Para trocar o WhatsApp, procure no index.html por: 5564992360895.
3. Para adicionar fotos novas, coloque os arquivos nas pastas dentro de assets/galerias.
4. Para editar pelo admin, clique em "Área admin" ou abra index.html?admin=1.

Observação sobre o admin:
- O login usa Supabase Auth.
- Os usuários precisam existir no Supabase Authentication com a senha definida por você.
- As edições do admin continuam sendo salvas no navegador via localStorage, com opção de exportar/importar JSON.
